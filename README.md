# Open-Source-Python-Package

A lightweight open-source package demo for practical NLP analysis, packaging, sentiment analysis, keyword extraction, batch inference, and stream-style chunk summaries.

## Repository contents
- `Open-Source-Python-Package.ipynb`
- `haddadnlpkit/`: package source exported from the notebook workflow.
- `pyproject.toml` and `setup.cfg`: package configuration files.
- `data/sample_texts.csv` and `data/sample_texts.txt`: sample inputs for batch testing.
- `assets/package_overview.png`: combined repository preview.
- `assets/batch_label_distribution.png`: batch label distribution figure.
- `requirements.txt`: lightweight dependency list.

## Main capabilities
- packaging-oriented NLP demo
- sentiment analysis
- keyword extraction
- batch inference
- stream-style chunk summaries
- project export for GitHub packaging

## Setup
Open the notebook in Google Colab and run it in a fresh runtime. The notebook builds and installs the local package within the active session.

## Usage
Use the single-text section for interactive analysis, the batch section for CSV or TXT processing, and the export section for packaging the generated source tree.
